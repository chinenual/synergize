# DX7 Algorithm to Synergy mapping

Convention:

```
1:6
```
means Synergy OSC 1, DX7 Op 6

## 1
```
               +---+
               |   |
             +-+-+ |
             |1:6| |
             +-+-+ |
               |   |
               +---+
               |
             +-+-+
             |2:5|
             +-+-+
               |
               |
               |
   +---+     +-+-+
   |5:2|     |3:4|
   +-+-+     +-+-+
     |         |
     |         |
     |         |
   +-+-+     +-+-+
   |6:1|     |4:3|
   +-+-+     +-+-+
     |         |
     +---------+
```
